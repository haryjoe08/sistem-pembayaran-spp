# 🎓 Sistem Informasi Pembayaran SPP

Sistem Informasi Pembayaran SPP merupakan aplikasi berbasis web yang dikembangkan untuk membantu sekolah dalam mengelola administrasi pembayaran Sumbangan Pembinaan Pendidikan (SPP) secara lebih efektif, cepat, dan terstruktur.

Aplikasi ini memudahkan petugas dalam melakukan pencatatan pembayaran, mengelola data siswa, serta menghasilkan laporan pembayaran secara digital.

## ✨ Fitur Utama

### 👨‍💼 Admin / Petugas

* Login dan autentikasi pengguna
* Mengelola data siswa
* Mengelola data kelas
* Mengelola data jurusan
* Mengelola data petugas
* Mengelola data SPP
* Mencatat transaksi pembayaran SPP
* Melihat riwayat pembayaran siswa
* Mencetak laporan pembayaran

### 👨‍🎓 Siswa

* Membayar tagihan secara online
* Melihat status pembayaran SPP
* Melihat riwayat pembayaran
* Mengakses informasi tagihan SPP
